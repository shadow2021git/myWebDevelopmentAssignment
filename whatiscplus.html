What is C++ ?

C++ is a general purpose, case-sensitive, free-form programming language that supports object-oriented, procedural and generic programming.

C++ is a middle-level language, as it encapsulates both high and low level language features.

C++ language is developed by Bjarne Stroustrup starting in 1979 at Bell Labs.

C++ runs on a variety of platforms, such as Windows, Mac OS, and the various versions of UNIX.

This C++ tutorial adopts a simple and practical approach to describe the concepts of C++ for beginners to advanded.
